## OCR识别服务列表


### <span id="ocr-common" >一、通用OCR识别</span>
| 接口方法 | 接口描述 |
|:-------------:|:-------------|
| [/ocr/common](#ocr-common) | 通用OCR识别 |


#### 1. 请求参数
| 参数名称| 必选 | 类型 | 说明 |
|:-------------:|:-------------|:-------------|--------------|
| file | true | MultipartFile |需要上传的文件|

---

#### 2. 调用样例
```
http://{ip:port}/ocr/common?
Body下选择form-data格式:
选择类型为:File
key:file
选择文件:选择要上传的文件
注:headers必需为空,否则上传会失败
```
---

#### 3. 返回结果
| 字段| 类型 | 说明 |
|:-------------:|:-------------|:-------------|
| status | String | 返回码 |
| errmsg | String | 返回码描述 |
| data | json | 数据 |

---
#### 4. JSON示例

* 请求成功数据格式：

```
{
    "status": 200,
    "errmsg": "成功",
    "data": " Java发编程的艺术java多线程编核心技术￥97.00包邮"
}
```
